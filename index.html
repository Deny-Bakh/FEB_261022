Галерея

Список базових команд для роботи з гітом в терміналі:

  • git init назва проєкту - створює новий локальний репозиторій із заданим ім'ям.
  • git clone url адреса - завантажує репозиторій разом із усією його історією змін.
  • git status - перераховує всі нові або змінені файли, які потребують фіксації.
  • git checkout -b branchname - створює нову гілку та переходить до неї.
  • git add . - розміщує нові файли та модифікації без видалення (у поточному каталозі та його підкаталогах).
  • git add * - переглядає все, але не файли, які починаються з крапки, і не видаляє файли, які було видалено з диска.
  • git commit -m "commit message" - фіксує проіндексовані зміни та зберігає їх в історію версій.
  • git commit -a - фіксує будь-які файли, які ви додали за допомогою git add, а також фіксує всі файли, які ви змінили з того часу.
  • git push origin branchname - надсилає гілку до свого віддаленого сховища, щоб інші могли її використовувати.
  • git branch Перелічує усі гілки у вашому репозиторії, а також повідомляє, у якій гілці ви зараз перебуваєте
  • git branch branchname - створює нову гілку.
  • git pull - завантажує історію з видаленого репозиторію і об'єднує її з локальної. pull = вибірка + об'єднання.
  • git merge branchname - щоб об’єднати іншу гілку у вашу активну гілку.
